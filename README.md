# Village Dominion

A village strategy and diplomacy simulator. Build your village, manage resources, interact with AI villages, form alliances, trade, and go to war.

- **Engine:** Godot 4.x  
- **Language:** GDScript

## How to run

1. Open **Godot 4.x**.
2. **Import** or **Open** the `godot-project` folder (the one containing `project.godot`).
3. Press **F5** to run the game.
